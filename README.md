# 🌿 GrowLeafy - Plant Tag Generator & Nursery Database

A lightweight, fast, and offline-friendly Nursery Database and Tag Printing application built with Streamlit and Supabase.

## Features

- 📊 Comprehensive Dashboard with statistics
- 🌱 Plant Database Management
- 🧪 Fertilizer Database
- 🐛 Insecticide Database
- 🛡️ Pesticide Database
- 🔍 Universal Search across all databases
- 🏷️ Professional A4 Tag Generator
- 📄 PDF Export with configurable layouts
- 🤖 AI Assistant (DeepSeek Integration)
- 📈 Reports Generation
- 📱 Mobile and Desktop Compatible
- 🌓 Light/Dark Mode

## Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **Database:** Supabase PostgreSQL
- **PDF Generation:** ReportLab
- **AI:** DeepSeek API
